#### 3.修改MyReceiver。
```java
public class MyReceiver extends BroadcastReceiver {

    @Override
    public void onReceive(Context context, Intent intent) {
        System.out.println("接收到了信息是："+intent.getStringExtra("msg"));
    }
}
```
